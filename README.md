# HelpConnectCar

Esse repositório contem o projeto para compor o desafio Happmobi

## Carga na base MongoDB

Para realizar a carga inicial da base, utilize o Postman onde ja contem todos os endpoints mapeados, realize o upload do arquivo `HELP CONNECT CAR.postman_collection.json`.
Através do endpoints `/veiculos` e possível carregar a base de carros.

Ou apenas rode o comando `npm run seed` dentro do projeto back-end para inicializar os dados iniciais dos veículos.

## Back-end start

`npm run dev`

## Front-end start

`ng serve --configuration development`
